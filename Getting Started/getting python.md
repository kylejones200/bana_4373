Python is the most popular computer programming language in the world and it is the standard languaged used for advanced analytics. The things we will do in this class go far beyond what is possible with Microsoft Excel. We will use a number of tools that are probably new for you. The point is to expose you to the actual tools that people use for business analytics. 

Accessing Python on the Cloud:

- AWS: you can sign up for a [[free account]{.underline}](https://aws.amazon.com/education/awseducate/) with AWS which will allow you to use SageMaker.
